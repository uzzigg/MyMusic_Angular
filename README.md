# Reproductor KT - Proyecto Angular (español)

Estructura minimal que cumple los requisitos solicitados:
- Angular 20 compatibles en package.json
- angular.json con root y defaultProject
- tsconfig con target ES2022
- Entidades de dominio en src/app/core/domain/entities
- Arquitectura hexagonal esbozada
- Servicios para Spotify (Client Credentials Flow)
- UI básica en español con dos rutas: /player y /buscar

Instrucciones:
1. `npm install`
2. `ng serve --ssl`
3. Abrir https://localhost:4200

Este repositorio es un *skeleton* completo listo para ejecutar y ampliar.
